# Intro

This is a simple app that funcions as a third party app for QQ API. 
1. UI. This is the user interface
2. Server. It implements the the callback, and functions as a gateway for QQ api calls. 

## Installation
Both are JS app. Assumed you have NODEJS insalled on your computer, do the following under each project
\`\`\`
npm install
\`\`\`

## UI
Launch the app, 
\`\`\`
npm start
\`\`\`

The app is running under port 3000

## Server
Launch the app,
\`\`\`
node index.js
\`\`\`

The app is running under port: 9000
